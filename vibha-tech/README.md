> **ViBha** = **Vi** (my sister Vika) + **Bha** (me, Bharath).
> Two people. One mission. Software that works for people who need it most.

ViBha Tech is a fintech company in Karur, Tamil Nadu, building purpose-built SaaS for microfinance institutions across India.

Products

- **ViBha Staff** — Flutter app for field loan officers (Play Store)
- **ViBha Admin** — Branch manager panel (Play Store)
- **ViBha Finan** — Finance tracking & daybook (in development)
- **Core Platform** — PHP backend, multi-tenant, OTP auth
